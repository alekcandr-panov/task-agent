# Task Agent — Telegram бот для автосбора задач

Автоматически фиксирует задачи из твоих сообщений в Telegram, хранит дедлайны и напоминает о них.

## Быстрый старт

### 1. Создать бота
- Открой @BotFather в Telegram
- Отправь `/newbot`, задай имя и username
- Скопируй токен

### 2. Узнать свой Telegram ID
- Напиши @userinfobot — он покажет твой ID

### 3. Установить зависимости
```bash
pip install -r requirements.txt
```

### 4. Настроить переменные окружения
```bash
cp .env.example .env
# Отредактируй .env — вставь BOT_TOKEN, OWNER_ID, ANTHROPIC_API_KEY
```

### 5. Запустить
```bash
python bot.py
```

---

## Как добавить бота в групповой чат

1. Добавь бота в нужный чат через «Добавить участника»
2. В настройках бота у @BotFather выключи Privacy Mode (`/setprivacy → Disable`)
   — это нужно чтобы бот видел все сообщения в группе, а не только команды

## Команды

| Команда | Действие |
|---------|----------|
| `/tasks` | Все открытые задачи, сгруппированные по приоритету |
| `/task_ID` | Детальная карточка задачи |
| `/done_ID` | Закрыть задачу |
| `/summary` | Статистика: открыто / просрочено / закрыто за неделю |
| `/help` | Справка |

## Как работает автосбор

```
Твоё сообщение
    ↓
Быстрый фильтр по ключевым словам (без API)
    ↓ (если похоже на задачу)
Claude анализирует текст + контекст чата
    ↓
Извлекает: название, описание, ответственный, дедлайн, приоритет
    ↓
Сохраняет в SQLite
    ↓
Присылает тебе подтверждение в личку
```

## Напоминания

- **Каждый час** — проверка просроченных задач и задач с дедлайном через 24 часа
- **09:00 каждый день** — утренний дайджест: просроченные, на сегодня, предстоящие

## Структура проекта

```
task_agent/
├── bot.py          — точка входа, все хэндлеры
├── agent.py        — извлечение задач через Claude API
├── database.py     — SQLite: CRUD операции
├── scheduler.py    — напоминания (APScheduler)
├── formatting.py   — форматирование сообщений
├── requirements.txt
├── .env.example
└── tasks.db        — создаётся автоматически
```
